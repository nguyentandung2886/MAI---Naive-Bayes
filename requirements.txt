# Pinned versions for reproducibility.
# WHY exact pins (==) and not ranges: the trained model is stored as a joblib
# .pkl of scikit-learn objects, and scikit-learn does NOT guarantee that a pickle
# saved under one version loads under another. If train-time and deploy-time
# scikit-learn differ, the app can silently break on load. Pinning ties the whole
# chain (local train -> git -> Streamlit Cloud) to one known-good version set.
#
# Chosen as a stable snapshot for Python 3.13: we deliberately avoid the newest
# majors (pandas 3.x, datasets 5.x, huggingface_hub 1.x) because the surrounding
# ecosystem tends to lag right after a major bump.

matplotlib==3.10.3
